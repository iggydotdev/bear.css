# bear.css
Bear (or Bare) CSS framework. Same philosophy as b0nes: simple and easy to understand in a day. 
```
bear.css/
├── src/
│   ├── bear.css          ← entry point, just @imports
│   ├── primitives.css
│   ├── tokens.css
│   ├── composition.css
│   └── properties/
│       ├── space.css
│       ├── type.css
│       └── color.css
├── dist/
│   └── bear.css          ← the one <link> file, generated
└── package.json
```
